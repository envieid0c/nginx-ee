# Nginx-EE 

Compile and install the latest nginx release with EasyEngine


![nginx-ee](https://raw.githubusercontent.com/VirtuBox/nginx-ee/master/nginx-ee.png)


-----
## Features
* Update Nginx to the Latest mainline release 
* Additonal modules
* TLS v1.3 Support

-----
